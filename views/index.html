Cloud Smart School Board

Whenever you need to lay out pictures, text notes, video and audio clips on a blank canvas, CSS Board can help you.

CSS Board is a browser based application. It is the right tool for you if you want to quickly put together a collage of your idea or concept, either for yourself or to share it with teammembers, clients or students. Changes are updated in realtime.

CSS Board is not meant for creating polished designs, but it is a good fit for:

  • Moodboards
  • Collages
  • Teaching (Virtual Blackboards)
  • Shared Whiteboards
  • Design Thinking
